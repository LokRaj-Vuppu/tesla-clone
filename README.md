# Tesla Website Clone ⚡

A pixel-perfect clone of the **Tesla homepage** built with **React**. Features smooth scroll animations, responsive design, Material UI components, and Redux Toolkit for state management.

> Portfolio project demonstrating modern React development and UI cloning skills.

---

## Features

- Responsive landing page design
- Smooth scroll animations using `react-reveal`
- Material UI (MUI) v5 components
- Redux Toolkit for state management
- Styled Components for CSS-in-JS styling
- Mobile-first responsive layout

---

## Tech Stack

| Technology | Purpose |
|-----------|---------|
| React 17 | UI library |
| Material UI v5 | Component library (@mui/material, @mui/icons) |
| Redux Toolkit | State management |
| styled-components | CSS-in-JS styling |
| react-reveal | Scroll animations |
| @emotion | CSS-in-JS runtime (MUI dependency) |

## Getting Started

### Installation

```bash
git clone https://github.com/LokRaj-Vuppu/tesla-clone.git
cd tesla-clone
npm install
```

### Run the App

```bash
npm start
```

Visit: **http://localhost:3000**

### Build for Production

```bash
npm run build
```
